# Security Policy

## Supported versions

This project is in active development. Security fixes are applied to the latest
`main` branch.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report privately by contacting the maintainer and include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

You can expect an initial response within 5 business days.
